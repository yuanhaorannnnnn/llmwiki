# HTML Effectiveness — AI 生成的 HTML 演示合集（离线镜像）

## 核心观点

Thariq 整理的一套 AI 生成 HTML 的实用案例合集。核心观点：HTML 是 AI 时代被低估的输出格式——Markdown 适合阅读，HTML 适合**体验**。20 个自包含 `.html` 文件，每个解决一种 Markdown 做不到的事：并排对比、动画演示、交互探索、可视化 diff、幻灯片。已离线镜像到 wiki。
