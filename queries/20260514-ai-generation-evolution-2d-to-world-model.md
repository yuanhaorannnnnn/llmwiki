# AI 生成式技术演进：从 2D 到世界模型的维度革命

## 核心观点

AI 生成技术的演进遵循一条从低维到高维的明确路径：2D图像生成 → 视频生成 → 3D生成 → 世界模型。每一次维度跃升不仅仅是数据量的增加，而是模型对物理规律的理解深度发生了质变。文章梳理了每一阶段的技术代表和核心突破。
