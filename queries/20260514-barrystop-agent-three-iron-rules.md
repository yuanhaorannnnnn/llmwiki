## 核心观点

Anthropic 经典文章 "Building Effective Agents" 的作者 Barry 在 AI Engineer Summit 做了升级版分享，把 agent 产品开发拆成三条铁律。和 2023 年底那篇文章相比，这次演讲更偏向产品决策层面——**什么时候该用 agent、什么时候不该用**的判断框架。

## 已知要点

从推文描述中可确认的内容：

1. **第一条铁律：别什么都用 agent 做**
   - 很多人一上来就想用 agent，但 agent 不是给所有任务用的
   - Barry 给了一份判断清单，帮助开发者区分"该用 agent"和"不该用 agent"的场景
   - 这与 Anthropic 文章的核心结论一致：**the simplest solution that works**
