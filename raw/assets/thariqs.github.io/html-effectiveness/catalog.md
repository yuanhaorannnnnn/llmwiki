# HTML Effectiveness — Layout Reference

When using `frontend-design` to create a page, check this catalog for a matching
page type. Read the corresponding HTML file to study its layout structure, color
palette, typography choices, and component patterns before designing.

**Base path**: `raw/assets/thariqs.github.io/html-effectiveness/`

## Usage

1. Identify the closest page type from the table above.
2. Read the HTML file to study its structure: layout grid, CSS variables, component patterns.
3. Use `frontend-design` to create the page, adapting the layout patterns observed.
