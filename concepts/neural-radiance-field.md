# Neural Radiance Field (NeRF) 技术栈

> NeRF（ECCV 2020）是一种基于逆渲染的神经场景表示方法。它的 MLP 不是"学习模型"，而是"连续的体数据查找表"——把 (x,y,z,θ,φ) 映射到体密度和颜色，通过体渲染方程合成像素，用多视角图像监督优化。这是理解 NeRF 所有"反常"性质（无跨场景泛化、几何质量差于视觉质量）的根本认知框架。

## 定义

NeRF 用**隐式 MLP** 表示三维场景。对于空间中的每个点和观察方向：

- MLP(x, y, z) → 体密度 σ（表示"这个位置有没有东西"，只取决于位置）
- MLP(x, y, z, θ, φ) → 颜色 c（表示"从这边看是什么颜色"，取决于位置+方向）

渲染一条射线时，在射线上采样 N 个点，通过体渲染方程（仅吸收模型）累积颜色：

C = Σᵢ Tᵢ · (1-exp(-σᵢδᵢ)) · cᵢ，其中 Tᵢ = exp(-Σⱼ₌₁ⁱ⁻¹ σⱼδⱼ) 是累积透射率

**为什么不是 CV 模型？**
- CV 模型的 MLP 学习"从数据中提取一般规律"（跨实例泛化）
- NeRF 的 MLP 学习"这个特定场景的空间属性"（场景内插值）
- NeRF 的"泛化"：在同一个场景中，对没有相机拍摄过的视角进行连续空间插值
- 所以 NeRF 需要在每个场景上重新训练，不能像分类网络那样训完即用

## 技术脉络

```
NeRF 核心框架 (ECCV 2020)
    │
    ├── 理解 NeRF 需要先理解的前置概念

    │     ├── 体渲染 — 将三维体数据可视化为 2D 图像（NeRF 的渲染后端）

    │
    ├── 工程加速
    │     ├── InstantNGP — 多分辨率哈希编码，训练从小时→分钟
    │     └── TensoRF — 张量分解替代 MLP，显式-隐式混合
    │
    ├── 质量改进
    │     ├── Mip-NeRF — 抗锯齿，锥形采样替代点采样

```

## 核心概念链：四步从零理解 NeRF

### 1. 逆渲染：从"画图"倒过来

- **渲染** = 3D → 2D：给定三维场景模型，计算指定视角下的图像（如游戏渲染）
- **逆渲染** = 2D → 3D：从多张二维图像反推三维场景信息
- 关键工具：可微分渲染——渲染管线可求导，用图像 loss 反向优化几何参数
- NeRF 属于逆渲染技术族，只是用 MLP 来参数化场景而非显式几何

### 2. 体渲染：为什么需要"沿射线积分"

- 体数据 = 三维空间每个离散格点的数值（如 CT 扫描的密度值）
- 体渲染 = 将体数据可视化为 2D 图像
- NeRF 采用"仅吸收"光学模型：光沿射线传播，被粒子吸收，不散射、不发光
- 核心方程：C = ∫ σ(t) · c(t) · e^(-∫ σ(s)ds) dt
- 直觉：射线上每个微元有"赋色因子"σ 和"颜色"c，离相机越远的微元贡献越小（被近处物体遮挡）
- 实际实现需离散化：均匀采样 → 逐段累加

### 3. Plenoxel：去掉神经网络，看清本质

**理解 NeRF 最关键的一步——先忘记 MLP。**

Plenoxel 用一个显式的 3D 网格存储场景：
- 每个网格顶点存：体积密度 σ + 球谐系数 SH（用于计算视角相关颜色）
- 任意采样点 → 三线性插值 → 得到 σ 和颜色
- 体渲染 → 像素颜色 → 与 GT 对比 → 直接优化网格顶点参数
- **不需要神经网络，就能做新视角合成**

NeRF 的 MLP 只是把显式网格换成了隐式映射：
- Plenoxel: `grid[x][y][z] → (σ, SH_coeffs)`
- NeRF: `MLP(x,y,z,θ,φ) → (σ, c)`
- MLP 的优势：天然连续性（不需要三线性插值）、参数更紧凑
- 显式表征优势：收敛快、推理快。代价：容易有伪影、存储大

### 4. NeRF：三个工程技巧

| 技巧 | 解决的问题 | 方法 | 直觉 |
|------|-----------|------|------|
| 位置编码 | MLP 偏向学习低频函数，渲染模糊 | γ(x) = [sin(2⁰πx), cos(2⁰πx), sin(2¹πx), ...] 傅里叶特征映射 | 给 MLP 一个"放大镜"，让高频细节也能被学到 |
| 层次化体采样 | 空白区域采样浪费算力 | 粗采样 N_c → 用粗结果权重分布 → 精采样 N_f → 两阶段 loss 相加 | 自动 Coarse-to-Fine，采样预算集中在有物体的区域 |
| 视角相关颜色 | 同一位置不同角度颜色不同（镜面反射） | σ 仅由位置(x,y,z)决定，c 由位置(x,y,z)+方向(θ,φ)决定 | 体密度回答"有没有东西"，颜色回答"这边看是什么色" |

## 学术分类体系

> 来自中科院计算所 2023 年 CSIG 综述《神经辐射场研究进展》，按**重建 → 处理 → 应用**三维分类。

### 1. 神经辐射场建模（重建）

| 子方向 | 核心问题 | 代表方法 |
|--------|---------|---------|
| 加速推理/训练 | NeRF 训练慢（小时级） | InstantNGP（多分辨率哈希）、TensoRF（张量分解） |
| 动态场景建模 | 场景随时间变化 | D-NeRF、HyperNeRF（变形场）+ 时间条件输入 |
| 城市场景建模 | 大规模、无界室外环境 | Block-NeRF（分块）、Mega-NeRF、Mip-NeRF 360（无界场景） |
| 跨场景/物体建模 | NeRF 只能训一个场景 | PixelNeRF、MVSNeRF（引入图像特征先验） |
| 稀疏视角建模 | 只有 2-3 张输入图 | RegNeRF、DietNeRF（正则化/语义约束） |
| 生成式模型 | 从噪声生成 3D 场景 | DreamFusion（SDS loss + 扩散先验） |

### 2. 神经辐射场处理（编辑）

| 子方向 | 核心问题 |
|--------|---------|
| 几何与纹理编辑 | 修改形状、移动物体、改变颜色纹理 |
| 材质和光照编辑 | 分解材质属性 → 重打光、换材质 |
| 图像处理 | 利用 NeRF 做去噪、超分、修复 |

### 3. 神经辐射场应用

| 应用方向 | 说明 |
|---------|------|
| 动态人体/人脸建模 | 从视频重建可驱动的人体 3D 模型 |
| 三维重建 | 从图像/视频重建高质量 3D 模型 |

## NeRF vs 3D Gaussian Splatting

| 维度 | NeRF | 3DGS |
|------|------|------|
| 表征类型 | 隐式（MLP 映射） | 显式（3D 高斯椭球） |
| 渲染方式 | 体渲染（沿射线积分采样） | 光栅化（tile-based，GPU 硬件加速） |
| 速度 | 训练小时级，推理秒级/帧 | 训练分钟级，推理 30+ FPS 实时 |
| 连续性 | 天然连续，无间隙 | 离散粒子，需处理空隙 |
| 可编辑性 | 困难（无法直接操作 MLP 权重） | 相对容易（移动/删除/添加粒子） |
| 物理精确性 | 高（光学模型） | 中等（近似光栅化） |
| 存储 | 几 MB | 几百 MB ~ GB |

本质关系：3DGS = 把 NeRF 的隐式 MLP 换成显式高斯粒子 + 把体渲染换成光栅化。**牺牲物理精确性，换取实时速度。** 两者都是逆渲染技术，用可微渲染+图像监督优化场景表征。

## 可迁移洞见

### "先去掉核心创新，看问题本身"

Plenoxel 的启发：当一个新的神经网络方法让人困惑时，先想"如果不用神经网络，这个问题本身应该怎么解"。NeRF 去掉 MLP 后就是 Plenoxel——一个显式 3D 网格 + 体渲染 + 直接优化的经典框架。MLP 只是实现手段，不是本质。同样的方法论适用于理解 Transformer、Diffusion 等——先问"没有这个架构，问题本身怎么解"。

### "泛化"有多种含义，不是所有 MLP 都是 CV 模型

NeRF 的 MLP 的"泛化"是空间连续插值（同一场景中 unseen 视角），不是跨实例泛化（见过的类别→没见过的类别）。混淆这两种泛化是误解 NeRF 的根源。在任何用神经网络的论文里，先搞清楚"它在什么意义上泛化"。

### "显式 vs 隐式"是一个连续谱

NeRF（纯隐式 MLP） → TensoRF（张量分解，半隐式） → Plenoxel（纯显式网格） → 3DGS（纯显式粒子）

没有绝对的"隐式更好"或"显式更好"。选择取决于你的瓶颈：存储→隐式，速度→显式，编辑→显式，连续性→隐式。NeRF 到 3DGS 的演进本质上是把连续谱往"显式+快速"方向推了一步。
