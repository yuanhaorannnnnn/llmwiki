# 神经隐式场

## 概述

神经隐式场是用神经网络表示连续空间函数的方法。给定一个空间坐标（以及可能的方向、时间等条件），网络输出该位置的目标值（颜色、密度、深度、距离等）。核心优势是"分辨率无关"——可以在任意分辨率/密度下采样。

## 核心问题

**显式 vs. 隐式表示之争**

- **显式表示**：体素网格、三角网格、点云、高斯椭球。优点是渲染快、可操控，缺点是内存占用随分辨率立方增长，拓扑固定。
- **隐式表示**：神经网络编码的连续函数。优点是任意分辨率、紧凑（参数远小于体素），缺点是渲染慢（需要逐点查询）。

**隐式场要解决的共性问题：**
1. 如何用有限参数表示无限分辨率的空间函数
2. 如何在查询点与训练点不一致时做合理插值
3. 如何加速逐点查询（采样策略、网格缓存、三线性插值）

## 主要应用

### 辐射场 → NeRF 家族
- **NeRF (2020)**：MLP 编码 (x, y, z, θ, φ) → (RGB, σ)。位置编码使网络学习高频细节。
- **Instant NGP (2022)**：多分辨率哈希网格 + 三线性插值，训练从天级加速到秒级。
- **3D Gaussian Splatting (2023)**：退回到显式表示（高斯椭球），用可微分光栅化替代隐式查询——更快、视觉效果更好。

### SDF → 表面重建
- **Occupancy Networks (2019)**：坐标 → 占据概率
- **DeepSDF (2019)**：有符号距离函数
- **PIFu/PIFuHD (2019-2020)**：像素对齐隐式函数，从单张图像重建穿衣服人体

### 深度场 → InfiniDepth
- **InfiniDepth (2026)**：坐标 → 深度值。把深度估计从离散网格解放到连续坐标查询。

### 其他
- **Video Implicit Fields**：坐标 (x, y, t) → 像素值，视频压缩/插值

## 关键概念

**"位置编码 = 降频补高频"**

SIREN/NeRF 的位置编码把连续坐标映射到高频正弦/余弦空间。为什么有效？神经网络天然学习低频平滑函数（谱偏置），位置编码把输入的高频成分降频后再输入网络。

**"哈希网格 = 显式存储 + 隐式查询"**

Instant NGP 把空间划分成多分辨率哈希网格，查询时三线性插值 + 小 MLP 解码。核心思想：大容量显式存储（哈希表）负责所有细节，小 MLP 只负责"拼接"和"调色"。

**"查询即推理"**

隐式场的计算模式：不在训练时生成整个场（那样会巨大），而是在推理时按需查询。每查询一个坐标点，运行一次网络前向。这决定了推理速度与查询点数成正比。

**"混合表示 = 取长补短"**

纯隐式场太慢 → 3DGS 回到显式但保留可微分性 → 更高灵活性。InfiniDepth 用了"ViT 显式特征金字塔 + 隐式解码器"的混合方案：全局特征用显式网格保存，局部深度用隐式解码器连续查询。

## 技术演进

```
显式体素 ──→ NeRF（纯隐式）──→ Instant NGP（哈希+隐式混合）
                                       │
                                       ├→ 3DGS（纯显式，可微分光栅化）
                                       │
                                       └→ InfiniDepth（特征金字塔+隐式解码器）
```

## 洞见

"隐式场真正的价值不是'隐式'，而是'连续'"——把离散样本转化为连续函数的能力。数据总是离散的（像素、体素），但物理世界是连续的。隐式场提供了一条从离散观测到连续函数建模的路径，应用遍及深度、辐射、距离、运动、语义等几乎所有空间计算机视觉任务。
