# 从图形计算到世界模型

> 陈宝权教授在 GAMES 2024 上的主旨报告，首次系统性地论证了计算机图形学（特别是物理仿真）与 AI 世界模型之间的深层关联。核心论点：**Simulation 是突破 Scaling Law 数据瓶颈的关键**——它能以物理规律为约束，生成带标签、可控制、多样化的训练数据；更进一步，可微分仿真（differentiable simulation）直接提供梯度监督信号，将仿真从"数据提供者"升级为"训练环境构建者"。

## 定义

### 世界模型的功能定义

学术界没有统一严格定义，但三个独立来源达成功能层面的共识：

| 来源 | 核心能力定义 |
|------|------------|
| Schmidhuber (LSTM 先驱) | Prediction（预测）+ Planning（规划决策） |
| Yann LeCun | 预测、推理、决策、规划——与人类大脑类比 |
| GPT-4o | 模拟、预测、规划、决策——通过学习和理解大量数据构建对现实世界的内部表示 |

**操作化定义**：以真实场景为输入 → 通过理解、分析、模拟、评价 → 输出符合物理规律的未来预测和决策。

### 图形仿真 ≡ Simulation ≡ 通向世界模型的路径

陈宝权的核心等式：**图形计算的核心目标 = 模拟真实世界 = Simulation**。在这个框架下，图形学从"画画的"变成了"造世界的"——不再只是渲染好看的图片，而是构建可以被 AI 作为训练环境使用的、物理上自洽的虚拟世界。

## 技术脉络

```
世界模型需要的核心能力
    │
    ├── 数据增广（Data Augmentation）
    │     └── 3D 数据极度稀缺（ShapeNet ~MB vs LAION ~PB）
    │          └── Simulation 生成可控、带标签、多样化的训练数据
    │
    ├── 训练环境（Training Environment）
    │     └── Real2Sim2Real 框架
    │          ├── Real → Sim: 传感器数据 → 数字孪生重建
    │          ├── Sim → Sim: 参数变化 → 生成丰富场景变体
    │          ├── Sim → Real: 在仿真中训练 → 部署到真实世界
    │          └── Real → Real: 有限真实数据的最终微调
    │
    ├── 正向仿真（Forward Simulation）
    │     └── 多物理场（刚体+软体+流体+磁）实时仿真
    │          └── 仿真越精确 → 学到的策略质量越高
    │
    └── 可微分仿真（Differentiable Simulation）
           ├── 逆向重建: 从真实数据反推物理参数
           ├── 优化设计: 梯度指导下的机器人形态/控制优化
           └── 物理代理模型: 不可微系统的可微替代 → 监督信号
```

## 核心概念详解

### 1. Real2Sim2Real —— Simulation 作为"数据压缩机"

Scaling Law 的对数曲线意味着：要持续降低 loss，数据量需要指数增长。3D 数据已经是 PB 级（文本/图像）vs MB 级（3D 模型）的量级鸿沟。

Real2Sim2Real 的核心逻辑：
```
Real 传感器数据（稀疏，无标签）
    → Real2Sim: 重建为数字化场景（NeRF/3DGS 在此介入）
        → Sim: 调参生成海量变体（视角/光照/动态/交互）
            → 大模型训练（数据瓶颈被仿真打破）
                → Sim2Real: 训练好的模型部署到真实世界
                    → Real2Real: 有限真实数据微调（弥合 sim-real gap）
```

### 2. 正向仿真——图形学的"硬核"任务

要成为世界模型的有效训练环境，仿真必须具备：

| 能力 | 当前状态 | 陈宝权团队的代表性突破 |
|------|---------|---------------------|
| 大规模软体实时变形 | 计算量巨大（四面体网格 + 体积不可压缩约束） | GPU 方法实现实时解算 |
| 参数化表面连续碰撞 | 长期存在的复杂问题 | 准确处理复杂参数化表面碰撞 |
| 流体-固体交互 | 单一物理无法覆盖 | 液体表面张力 + 流固耦合模型 |
| 磁流磁软体 | 国际领先 | Siggraph 2024：100倍性能提升（100s→1s/帧 @512³） |

### 3. 可微分仿真——从"数据提供者"到"训练环境构建者"

这是全文最重要的技术方向。可微分仿真从根本上改变了仿真和 AI 的关系：

**传统仿真**：提供训练环境 → RL agent 在其中试错。问题：长时间尺度下奖励序列稀疏，策略优化容易陷入困境。

**可微分仿真**：仿真过程本身可求导 → 梯度直接回传给策略网络 → 监督学习范式替代纯 RL。

```
不可微仿真:  Sim → 试错 → 奖励信号 → 策略更新（慢，不稳定）
可微分仿真:  Sim → 梯度回传 → 直接监督 → 策略更新（快，稳定）
```

关键工作举例：
- **逆向物理参数拟合**（王滨）：从真实荷叶晃动数据 → 可微分仿真优化 → 恢复完整物理参数 → 该荷叶在任何条件下的动态都可准确预测
- **PINN 流体重建**（楚梦渝）：基于物理知悉网络的可微性，从真实流体数据重建物理场
- **磁软体机器人控制**（陈旭雯）：可微仿真提供梯度 → 外磁场参数优化 → 机器人爬坡/越障
- **物理代理模型**（刘利斌）：仿真环境不可微 → 训练一个可微的代理模型 → 首次实现大规模运动控制的生成式建模

### 4. 物理代理模型（Physics Surrogate Model）

当完整仿真无法可微化时的变通方案：

```
真实物理仿真（不可微）
    → 训练一个可微的神经网络代理
        → 代理模型近似仿真行为且可微
            → 通过反向传播提供物理监督信号
```

刘利斌团队用这个方法首次实现了大规模运动控制的生成式建模——压缩几十小时的多样化运动数据，训练出高度泛化的动作控制模型。

## 可迁移洞见

### "数据不够 → 仿真来凑"是 Scaling Law 的内部逻辑

Scaling Law 说 loss ∝ 1/log(data)。这意味着要持续降低 loss，数据必须指数增长。真实世界的数据永远追不上指数曲线——仿真提供了唯一可持续的数据源。这不是"锦上添花"的工程优化，是 Scaling Law 继续生效的前提条件。

### "仿真不是为了画得像，是为了物理上对"

陈宝权的报告反复强调：Simulation 越精确 → 学到的策略质量越高。这和我们之前的 PhysX-Anything 概念页形成呼应：**物理正确性是比视觉正确性更高的标准**。只有物理自洽的仿真才能产出可迁移的策略。

### "可微分仿真 = 从 RL 到监督学习的范式转换"

RL 的不稳定性和收敛困难是已知瓶颈。让仿真可微——让整个物理世界能反向传播——把"试错"变成"算梯度"。这不是工程技巧，是训练范式的层级跃迁。

### "Real2Sim2Real 是连接两个世界的桥梁"

没有人有足够的真实数据来训练世界模型。也没有人能只靠仿真数据来保证泛化。Real2Sim2Real 的闭环——真实采样 → 仿真扩展 → 部署微调——是目前唯一自洽的路径。
