# 4D 场景分解

## 概述

4D 场景分解是从多视角视频中重建动态三维场景，并将其拆分为独立实例（人、物体、背景）的技术。与静态场景重建不同，4D 分解需要处理时间维度上的运动、遮挡和实例一致性。

核心挑战：
1. **运动建模**：物体在时空中移动，重建表示必须支持变形或流动
2. **实例一致性**：同一物体在不同时间、不同视角被识别为同一个实例
3. **遮挡处理**：物体互相遮挡时，深度排序和可见性判断更复杂

## 核心问题

**旧方法的困境：**

- **视频追踪器依赖**：SA4D 等先用 2D 视频追踪（如 DEVA）做实例分割，再抬升到 4D。追踪器在长序列、遮挡、重识别场景下不稳定，错误直接传播到 3D。
- **Deformable 表示局限**：SADG 用 canonical 空间 + 变形场，但大位移、非刚性运动下重建质量差，特征关联断裂。
- **多视角一致性难**：不同视角的分割标签需要人工对齐，自动化程度低。

**新思路（Split4D）：**

不用视频追踪器，也不用 canonical 变形。而是：
1. 用**短寿命高斯粒子**（FreeTimeGS）在时空中自由流动
2. 每个粒子带**可学习特征向量**，通过对比损失在 2D 分割图上学习
3. **流式采样策略**：按时间顺序训练，让特征像水流一样稳定传播
4. 粒子的**线性运动函数**隐式建立时间对应关系

## 技术演进

### 动态场景表示
- **4DGS (Yang et al., 2024)**：4D 高斯，时空统一表示
- **FreeTimeGS (Wang et al., 2025)**：短寿命高斯粒子 + 线性运动
- **Split4D (Hu et al., 2025)**：在 FreeTimeGS 上加特征向量 + 对比学习，实现 4D 分割

### 场景分解策略
- **OmniRe (2025)**：城市场景分解（静态/刚体/SMPL/变形），但不做实例分割
- **SA4D (2024)**：视频追踪 + 4D 特征场，依赖追踪器质量
- **SADG (2024)**：Deformable 4DGS + 对比学习，重建质量受限
- **Split4D (2025)**：无追踪器、无 canonical 空间，粒子流动自追踪

## 关键概念

**"粒子流动 = 时间追踪"**

传统方法显式计算"帧 A 的实例 1 = 帧 B 的实例 1"。Split4D 不这么做——高斯粒子自带运动函数 µx(t) = µx + v·(t-µt)，粒子的轨迹就是隐式追踪路径。同一个粒子在不同时间出现，它的特征自然要保持一致。

**"短寿命粒子接力"**

复杂运动由多个短寿命粒子分段覆盖。每个粒子只负责一小段线性运动，但通过流式采样策略，特征在相邻帧之间稳定传播，远距离帧通过中间帧逐步对齐。

**"流式采样 = 渐进式收敛"**

随机采样像"同时学所有帧"，远距离帧特征容易各奔东西。流式采样像"一帧一帧学"，先学好 t=1，再带着知识学 t=2，特征像水流一样传播，不会突变。

**"对比学习跨时间"**

虽然每帧的分割图是独立的（SAM 逐帧分割，ID 不跨帧），但粒子会动。粒子 A 在 t=1 属于球员 1，在 t=2 飘到球员 1 的新位置——粒子 A 的特征在 t=1 和 t=2 都参与对比损失，自然把时间维度连起来。

## 洞见

"追踪的本质是建立时间对应关系，但对应关系不一定需要显式计算"——当表示本身包含运动（如速度、出生时间），时间一致性可以由物理运动自然保证，不需要额外的追踪模块。

这和人类视觉的"物体恒常性"类似：我们看到一个球飞出去，不需要显式追踪它的 ID，只需要知道"这个运动的视觉特征属于同一个物体"。
